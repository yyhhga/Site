# Setup the project

1. install dependencies.

   `npm i`

2. Run your dev instance

   `npm run dev`

##### Storybook

To run Storybook:

`npm run storybook`

# Deployment

This project is deployed with Cloudflare.
Manage Production builds in Cloudflare admin console.
